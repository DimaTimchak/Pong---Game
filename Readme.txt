
Гра написана на мові С++. Використовував інструменти та функції цієї мови

В даній грі Створено декілька класів. Базовий клас "Entity" в якому реалізовано підгрузку текстур для об'єктів. Від нього наслідуються такі класи як: "Background" (реалізовує задній фон), "Paddle_player" (реалізовує загрузку текстури для гравця, та керування), "Padle_ai" (реалізовує загрузку текстури для бота, та описує логіку бота), "Ball"(загружає текстуру для нашого м'яча, та реалізовує функціонал м'яча який літає від одного гравця до іншого та слідкує кому з гравців добавити очки) Також створений клас "Score" який відповідає за рахунок в грі.
	Для малювання гри використав графічну бібліотеку SFML. 
Створив відповідні класи які відповідають за малювання Головного меню, та самої гри. В файлі Game_state.h створені та реалізовані 2 класи. "Game_state" який відповідає за викликом функцій для створення та малювання вікна та інших об'єктів. "Tiny_state" клас в якому оголошені віртуальні методи для малювання вікта та об'єктів. "Tiny_state" базовий клас для класів  "Main_game" та 'Main_menu'. 
"Main_game" підгружає потрібні шрифти, реалізовує сам процес гри,  виділяє на кучі всі об'єкти та викликає їхні методи. 
"Main_menu" займайється малюванням та функціоналом головного меню.

З контейнерів STL використовував клас Vector для зберігання координат об'єктів
реалізацію можна побачити в Класі "Entity" де створювався сам вектор, та в інших класах де він використовувався. 
З розумних вказівників використовував "Shared_ptr" за допомогою нього вікно створював на кучі, та відслідковував його знищення. Реалізацію можна побачити в головній функції main та в класі Game_State
